# REQUIRED: Add these to your .env.local file

## For Admin Dashboard to work:

# Your Telegram ID (get from @userinfobot)
ADMIN_TELEGRAM_IDS=YOUR_TELEGRAM_ID_HERE

# Port for admin dashboard (optional, default is 3000)
ADMIN_PORT=3000

## Example .env.local structure:
# BOT_TOKEN=your_bot_token_here
# POSTGRES_URI=postgresql://user:pass@localhost:5432/dbname
# ADMIN_TELEGRAM_IDS=123456789
# ADMIN_PORT=3000

## Quick Setup:
1. Get your Telegram ID: Message @userinfobot on Telegram
2. Copy the numeric ID (e.g., 123456789)
3. Add ADMIN_TELEGRAM_IDS=123456789 to .env.local
4. Restart: node start-all.js
5. Visit: http://localhost:3000/admin/login

## Testing Login:
1. Open http://localhost:3000/admin/login
2. Enter your Telegram ID
3. Click "Get Login Code"
4. You'll see a 6-digit code (e.g., 456789)
5. Message your bot: /admin_login 456789
6. Bot will verify and redirect you to dashboard
